# :open_file_folder: [_direnv_](https://github.com/direnv/direnv)
> Unclutter your `.profile`

Environment switcher plugin for [Wahoo][Wahoo] / [fish](fishshell.com).

## Install
> _Note_: You need to have `direnv` already installed.

```fish
wa g direnv
```
